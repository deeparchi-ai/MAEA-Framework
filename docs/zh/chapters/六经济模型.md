## 六、经济模型

### 月预算结构

| 项目 | ¥/月 | 占比 |
|---|---|---|
| API 调用费 | 16,900 | 80.9% |
| 阿里云基础设施 | 2,500 | 12.0% |
| 管理预留 | 1,500 | 7.2% |
| **合计** | **20,900** | **100%** |

### 日运营成本

单个 Agent 日均 API 费用在 ¥27-117 之间，全集群日均 ¥743。其中战略分析（¥127/日）和架构治理（¥120/日）因双模型交叉验证，成本最高——这也是质量和安全的必要投入。

### Token 经济

MAEA 的预算模型将 Token 视为可计量、可预算、可审计的资源。每个 Agent 在月预算内运行，超额自动熔断。这和企业管理中的「部门预算」逻辑一致——资源约束推动优先级判断。
