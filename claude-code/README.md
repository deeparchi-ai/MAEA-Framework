# MAEA for Claude Code

将 MAEA 治理框架作为 Claude Code Agent 使用——30 秒内拥有 AI 团队的治理能力。

## 安装

```bash
# 安装治理 Agent
cp maea-architect.md ~/.claude/agents/
```

## 使用

在 Claude Code 中激活：

```
激活 MAEA 架构治理模式。我有 [N] 个 AI Agent，现在要做 [任务描述]。
帮我推荐和协调。
```

## 可用的 Agent

| Agent | 职责 |
|-------|------|
| `maea-architect` | 任务路由 + 冲突检测 + 质量验证 |
